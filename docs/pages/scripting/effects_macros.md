
## Effects Names

They are used to access the effect.

Example:
```gml
FF_BLOOM
```
So I can indicate which effect I'm "accessing". This can be useful in the following case, for example:
```gml
ppfx_id.SetEffectEnable(FF_BLOOM, false);
```
In the example above, I turned off the "Bloom" effect.

## Effects Parameters

Use them to indicate which effect variable to modify. They start with a capital `PP_`, to facilitate code-completion (intellisense).

Example:
```gml
ppfx_id.SetEffectParameter(FF_BLOOM, PP_BLOOM_KNEE, 1.5);
```
In the example above, I am modifying the bloom knee to 1.5, using PP_BLOOM_KNEE.
